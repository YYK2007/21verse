# Support

Use GitHub issues for:

- Unity setup problems.
- Bugs in the included scenes.
- Documentation fixes.
- Validation failures.
- Accessibility or learning-design improvements.

For security concerns, follow `SECURITY.md`.

- `README.md`
- `docs/game-design-summary.md`
