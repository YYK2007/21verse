# Support

21Verse is currently in private staging for a future open-source release.

## Before Public Release

Use the private repository issues and project channel for support. Do not post private Google Drive, NAS, testing, research, financial, investor, partner, outreach, or credential material into public channels.

Useful starting points:

- `README.md`
- `docs/public-release-runbook.md`
- `docs/release-readiness.md`
- `docs/inventory/release-audit.md`
- `CONTRIBUTING.md`
- `SECURITY.md`

## After Public Release

Once the repository is public, use GitHub issues for:

- Unity setup or validation problems
- Documentation fixes
- Public-safe asset replacement work
- Bugs in the included Unity scenes

Do not use public issues for vulnerability reports or private/sensitive materials. Follow `SECURITY.md` for those.
