# Code of Conduct

21Verse aims to be a respectful, educational, and accessibility-minded project.

## Expected Behavior

- Be respectful and constructive in issues, pull requests, reviews, and discussions.
- Assume good intent, but accept feedback when something causes harm or confusion.
- Keep feedback focused on the project, code, documentation, design, and release readiness.
- Be careful with language around disability, education, research participants, students, and families.
- Protect private research, testing, financial, partner, investor, outreach, and personal information.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private files, credentials, user data, research data, or partner material.
- Pressuring maintainers to make the repository public before the release blockers are resolved.
- Ignoring licensing, attribution, or redistribution concerns for third-party assets.

## Reporting

While this repository is private, report conduct concerns directly to the repository owner through the private project channel.

After public release, this section should be updated with the preferred public contact path.

## Enforcement

The repository owner may edit, hide, or remove content and restrict participation when behavior conflicts with this code of conduct or the release-safety rules in `SECURITY.md`, `CONTRIBUTING.md`, and `docs/release-readiness.md`.
