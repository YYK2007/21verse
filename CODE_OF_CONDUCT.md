# Code of Conduct

21Verse aims to be a respectful, educational, and accessibility-minded project.

## Expected Behavior

- Be respectful and constructive in issues, pull requests, reviews, and discussions.
- Keep feedback focused on the project, code, documentation, design, and learning experience.
- Use careful language around disability, education, students, and families.
- Accept correction when something causes harm or confusion.
- Respect licensing, attribution, privacy, and consent.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Dismissive or demeaning language about disability or learning differences.
- Publishing credentials, personal data, participant data, or sensitive records.
- Ignoring licensing, attribution, or redistribution requirements for third-party assets.

## Reporting

Report conduct concerns to the repository owner or project maintainers. Maintainers may edit, hide, or remove content and restrict participation when behavior conflicts with this code of conduct.
